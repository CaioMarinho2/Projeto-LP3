# Projeto-LP3
Projeto-avaliação desenvolvido para a disciplina de Lingiagem de Programação 3 da Universidade Federal Rural do Rio de Janeiro. Este repositório contém o desenvolvimento de um sistema acadêmico criado como parte da disciplina Linguagem de Programação 3. O objetivo principal é aplicar conceitos de Java, JavaFX, DAO, MVC, conexão com banco de dados e boas práticas de programação.
